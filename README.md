## ✨ What DPWriter Does

DPWriter helps you turn rough ideas into usable text.

- Plan story paths before you write
- Try different branches for a scene
- Keep track of ideas in one place
- Build drafts step by step
- Work on creative writing with less rework

It fits well for stories, scripts, scenes, and idea notes.

## 🧭 How to Use It

### 1. Open the app

Launch DPWriter from the Start menu, desktop, or the folder where you installed it.

### 2. Start a project

Choose a new project name. Use a short name that helps you find it later.

### 3. Add your idea

Enter a prompt, scene idea, or topic. Keep it simple at first.

### 4. Build branches

Create more than one path for the same idea. This helps you compare choices before you write the final version.

### 5. Review the output

Read each branch and pick the one that fits your goal. You can edit and refine it as you go.

### 6. Save your work

Save often so you do not lose your drafts.

## 📝 Best Way to Begin

If you are new to DPWriter, start with a short scene.

Example:

- Main idea: a lost letter
- Branch 1: the letter reaches the wrong person
- Branch 2: the letter changes a family secret
- Branch 3: the letter was never lost

This makes it easier to see how branching works before you move to larger projects.

## 🔧 Common Tasks

### Create a new draft
Use a blank project and enter one scene idea.

### Compare two versions
Write two branches with different tones, such as calm or tense.

### Revise a scene
Pick one branch, then edit the text until it matches your goal.

### Organize ideas
Keep related notes in one project so you can find them later.

## ⌨️ Tips for Better Results

- Start with one clear idea
- Use short prompts
- Keep branches focused on one change
- Save your work after each round
- Read each version out loud
- Pick the branch that feels natural
